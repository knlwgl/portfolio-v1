---
title: "Presidential Speech Classifier"
startedOn: "2024-06"
summary: "Trained several models using presidential speeches in order to classify any text to a president."
---

**Technologies:** Python, Tensorflow, NLTK, Gensim, scikit-learn, pandas

- Engineered a custom Transformer encoder block layer extending the Keras Layer class for binary classification.
- Trained models using Transformer and LSTM architecture to label text based on U.S. presidential speeches.
- Examined several classifier architectures, by testing on various real and fictional speeches, to analyze performance.
