**Technologies:** Java, Git

- Created a modular command-line game with two modes, using the Model-View-Controller design pattern.
- Built client-server communications for multiplayer mode using JSON and the Jackson Library.
- Developed a simple AI for local play that uses a hunt-target algorithm to provide challenging gameplay.
