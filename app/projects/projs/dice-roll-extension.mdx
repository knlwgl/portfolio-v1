---
title: "Dice Roller Chrome Extension"
publishedAt: "2023-12"
summary: "Created an interactive Chrome extension for rolling dice in virtual board games."
---

**Technologies:** TypeScript, React.js, CSS, HTML

- Developed an aesthetically pleasing chrome extension that allows users to roll dice for use in virtual board games.
- Incorporated CSS to animate the rolling of the dice and to replicate the feeling of rolling a real die.
- Enabled users to change the number of dice, each with a discrete roll, allowing adaptation to different situations.
